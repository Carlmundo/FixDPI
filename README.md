# FixDPI

## About
Fix DPI scaling issues in games and applications that don’t display correctly on high-DPI screens.

This tool adds a **“Fix DPI”** option to the right-click context menu for `.exe` files.  
It automatically sets the same compatibility option as:

> Properties → Compatibility → Change high DPI settings → Override scaling behavior → Application

This helps fix blurry or incorrectly scaled applications on high-resolution displays.
